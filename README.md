# photo-variator

Create a software that will take a singular photo and be able to create as many variations of this photo or video that all have different data, in order to allow it to be posted multiple times but seem different each time

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/photo-variator.git
cd photo-variator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
